# Photos go here

Drop your real photos in this folder, then update the relevant HTML file.

0. **Founder / team photos (About page)** — in `about.html`, find the
   3 `.team-photo` divs and replace each `<div class="frame-ph">...</div>`
   with:
   ```html
   <img src="assets/images/founder-1.jpg" alt="Founder name">
   ```
   Recommended size: square, at least 600×600px.



1. **Hero photo** — find this block near the top of `index.html`:
   ```html
   <div class="frame-ph">
     <div class="frame-tag">No. 001</div>
     ...
     <div class="ph-label">Your photo here</div>
   </div>
   ```
   Replace the whole `<div class="frame-ph">...</div>` with:
   ```html
   <img src="assets/images/hero.jpg" alt="Mohsin Studio wedding photography">
   ```

2. **Gallery photos** — these 9 are generated by `js/script.js` (look for
   the `galleryItems` array near the top). Each entry has a category
   (Mehndi/Baraat/Walima/Portraits/Events) used by the filter buttons.
   Easiest approach: name your photos `gallery-01.jpg` through
   `gallery-09.jpg`, drop them in this folder, and change this line
   inside the `.forEach(([category, caption], idx) => { ... })` block:
   ```js
   <div class="frame-ph"> ... </div>
   ```
   to:
   ```js
   <img src="assets/images/gallery-${num}.jpg" alt="${caption}">
   ```
   Tip: order your photos in the array to match real categories — e.g.
   put your best Mehndi shots first if most entries are `"Mehndi"`.

**Recommended sizes:** hero photo ~1200×1500px, gallery photos ~800×800px,
compressed as `.jpg` (quality ~80%) so the site stays fast to load.
